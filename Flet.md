  
Flet — фреймворк для создания кроссплатформенных приложений на Python. Предоставляет [[Flutter]] компоненты для разработки кроссплатформенных приложений на Python (разработчики обещают расширять список поддерживаемых языков).

Не использует SDK, не компилирует код Python в код Dart и весь UI отображается с помощью встроенного Web сервера.

— `pip install flet`
